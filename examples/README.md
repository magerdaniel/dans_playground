# Examples

This directory contains example projects and proof-of-concepts for AI on BI applications.

## Example Project Structure

Each example should be self-contained with:
- Clear README explaining the purpose
- Requirements/dependencies file
- Source code with comments
- Sample data (if applicable)
- Instructions to run

## Planned Examples

### 1. Basic BI API Connection
- Connect to a BI platform API
- Authenticate and retrieve metadata
- List available reports/dashboards

### 2. Natural Language Query Interface
- Accept natural language questions
- Convert to SQL or API calls
- Return results in readable format

### 3. Automated Insight Generator
- Analyze business metrics
- Generate written insights using LLM
- Create summary reports

### 4. Data Quality Monitor
- Check data completeness
- Identify anomalies
- Alert on issues

### 5. Interactive Dashboard Chatbot
- Query dashboard data conversationally
- Explain visualizations
- Provide recommendations

Add your experiments and projects here. Each should demonstrate a specific concept or technique.
