### Adding Resources

When adding files:
- Use clear, descriptive names
- Include a brief description
- Respect licensing and data privacy
- Don't commit large files (use Git LFS if needed)

## Notes

Avoid committing:
- Sensitive data or credentials
- Large datasets (>10MB)
- Proprietary information
